# Customer Segmentation Project

This project uses Machine Learning to segment customers into different groups based on their purchasing behavior and spending patterns. The goal is to help businesses understand customer categories such as high-value, medium-value, and low-value customers for better decision-making and targeted marketing.

## Features
- Data preprocessing and cleaning  
- Customer segmentation using K-Means Clustering  
- Optimal cluster evaluation using Silhouette Score  
- Data visualization for customer groups  
- Insights and interpretation of customer segments  

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

## Project Outcome
This model identifies customer segments that can support:
- Personalized marketing  
- Customer retention strategies  
- Business growth decisions
